⚙️ Setup Bot Telegram - Panduan Lengkap

🤖 1. Setup BOT_TOKEN

Langkah-langkah:

  1. Buka Telegram dan cari @BotFather
  2. Ketik /newbot untuk membuat bot baru
  3. Berikan nama untuk bot Anda (contoh: "My Hosting Bot")
  4. Berikan username untuk bot (contoh: "myhostingbot")
  5. Copy token yang diberikan BotFather
  6. Di Replit: Buka tab "Secrets" → Klik "+ New Secret"
  7. Key: BOT_TOKEN | Value: paste token dari BotFather

⚠️ Penting: Jangan share token bot ke siapapun!

🔗 2. Setup WEBHOOK_URL

Untuk Replit (Recommended):

  1. Pastikan bot server sudah running (klik tombol "Run")
  2. Copy URL repl Anda: https://[repl-name].[username].repl.co
  3. Tambahkan /webhook di akhir URL
  4. Di tab "Secrets" → "+ New Secret"
  5. Key: WEBHOOK_URL | Value: https://[repl-name].[username].repl.co/webhook

Contoh URL Webhook untuk platform lain:

  • Vercel: https://your-project.vercel.app/webhook
  • Railway: https://your-app.railway.app/webhook
  • Render: https://your-app.onrender.com/webhook
  • Heroku: https://your-app.herokuapp.com/webhook

🚀 3. Aktivasi Webhook

Setelah BOT_TOKEN dan WEBHOOK_URL sudah diset:

  1. Buka tab "Console" di Replit
  2. Ketik perintah: python webhook_setup.py
  3. Tunggu sampai muncul pesan "✅ Webhook berhasil diset!"
  4. Test bot di Telegram dengan mengirim /start

🛠️ Troubleshooting

Bot tidak merespon?

  • ✅ Pastikan repl sedang running (lampu hijau di tab)
  • ✅ Cek BOT_TOKEN di Secrets sudah benar
  • ✅ Cek WEBHOOK_URL sudah sesuai format
  • ✅ Jalankan ulang webhook_setup.py
  • ✅ Test endpoint: [your-repl-url]/health

📊 Informasi Teknis

🔗 Endpoints tersedia:

  • / - Dashboard analytics (halaman ini)
  • /webhook - Endpoint untuk Telegram webhook
  • /health - Health check status server
  • /test - Test konfigurasi bot
  • /api/stats - API statistik bot (JSON)

📊 Database: SQLite (bot_logs.db)

🚀 Server: Flask Development Server pada port 5000
